== Installation ==

= Method 1 =

Use WordPress Add New Plugin feature and search Bold Page Builder 

[youtube https://www.youtube.com/watch?v=FYFGEE5LFvg]

= Method 2 =

Download the archive [here](https://downloads.wordpress.org/plugin/bold-page-builder.zip) and:

1. Unzip the archive on your computer
2. Upload bold-builder directory to the /wp-content/plugins/ directory
3. Activate the plugin through the Plugins menu in WordPress

= How to import demo content =

1. Once you have installed and activated Bold Page Builder, you can download demo import package here: http://bold-themes.com/local/bold_builder/demo_data.zip.
2. Please unpack demo_data folder and upload it via FTP to your plugin's export_import folder (the server path is usually: \wp-content\plugins\bold-page-builder\export_import)
3. You should have \wp-content\plugins\bold-page-builder\export_import\demo_data\ with a number of xml files in it.
4. If everything went well, once you refresh your wp-admin console you should have Bold Builder Import item in Tools menu of your wp-admin

To import demo go to Tools > Bold Builder Import and select a demo. Make sure that appropriate theme is activated (twentysixteen or twentyseventeen).
